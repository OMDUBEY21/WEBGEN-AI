# WebGen AI

WebGen AI is an AI-powered platform that transforms natural language business ideas into fully functional, multi-page websites. It automatically generates frontend assets, packages project deliverables, and deploys production-ready websites with a single prompt.

## Key Features

- Natural language to website generation
- Multi-page responsive website creation
- Automated frontend asset generation
- Website editing and regeneration
- ZIP packaging of project files
- One-click deployment to Vercel
- Live production URL generation
